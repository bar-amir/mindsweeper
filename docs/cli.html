CLI¶

The CLI module offers command line iterface for consuming the API of Minesweeper, querying its database and printing the results to the console.

CLI¶

python -m mindsweeper.cli¶

python -m mindsweeper.cli [OPTIONS] COMMAND [ARGS]...

get-result¶

python -m mindsweeper.cli get-result [OPTIONS] USER_ID SNAPSHOT_ID RESULT_NAME

-p, --port <port>¶
