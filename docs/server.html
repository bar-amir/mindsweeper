Server¶

The Server module offers methods for running the server, and an API for uploading messages to it.

mindsweeper.server.run_server(host='127.0.0.1', port=8000, publish=None)¶

Listen on host:port and pass received messages to publish.

mindsweeper.server.upload(msg, publish)¶

Receive a message in a format readable by the server, and make it ready to be sent to the message queue: Messages with types mentioned in any parser’s msg_types, will change their status to ‘unparsed’. Other messages will change their status to ‘ready’. For messages with large amount of data (like images), their data should be saved to a reasonable path under the ‘media’ folder.

mindsweeper.server.upload_api()¶

API for uploading messages to the server. Send a BSON decoded message to /upload.
