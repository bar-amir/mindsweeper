Auxiliary functions¶

A collection of small helpful functions.

mindsweeper.utils.aux.find_parser(parser_name)¶
Returns

A tuple of the parser-function parser_name, found in ‘parser_`parser_name`.py’, and its msg_types, i.e. the messages this parser-function handles.

Return type

(function, set)

mindsweeper.utils.aux.get_interesting_types()¶

Return a set of all message types that at least one parser is interested in.

mindsweeper.utils.aux.get_parsers()¶

Return a list of tuples (parser_name, msg_types), where msg_types is a set of message types that parser_name is interested in.

mindsweeper.utils.aux.get_parsers_list()¶

Return a list of available parsers according to their filenames.
